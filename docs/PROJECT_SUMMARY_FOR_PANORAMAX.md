# Short project summary for Panoramax debugging

Foto Geo-Tool is a browser/PHP tool for restoring geodata in photos and video frames.

The main use case is:
- photos or videos from WhatsApp or similar apps lose EXIF/GPS metadata,
- the user manually sets GPS position, altitude, heading and description,
- the tool saves prepared JPEG images,
- the PHP script uploads them to Panoramax.
